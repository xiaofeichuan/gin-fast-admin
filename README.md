# gin-fast-admin

```bash
# 进入前端
cd web

# 安装依赖
cnpm install

# 运行项目
cnpm run dev

# 打包发布
cnpm run build
```

```bash
# 进入服务端
cd server

# 初始化依赖
go mod init

# 启动项目
go run main.go
```
